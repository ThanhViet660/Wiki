✨ Tính năng chính

Tất cả tính năng được tích hợp trong VietCraftUpgrade

🔄 Hệ thống trao đổi vật phẩm

Hệ thống cốt lõi cho phép người chơi trao đổi nguyên liệu để nhận vật phẩm mới theo công thức được admin cấu hình.

  • Hỗ trợ cả MMOItems (SWORD, BOW, ARMOR, MATERIAL...) và Vanilla Items (DIAMOND, IRON_INGOT...)
  • Tổ chức theo cấu trúc phân cấp: Group → Category → Recipe → Ingredient
  • Mỗi recipe có kết quả (Result) và danh sách nguyên liệu (Ingredients)
  • Hỗ trợ trao đổi 1 lần hoặc nhiều lần liên tiếp
  • Hiển thị trạng thái nguyên liệu (đủ/thiếu) theo thời gian thực
  • Menu GUI đẹp mắt với MiniMessage formatting
⬆️ Nâng cấp vật phẩm

Tự động tìm công thức nâng cấp dựa trên hệ thống level. Plugin phân tích ID vật phẩm để xác định level hiện tại và tìm recipe cho level tiếp theo.

🔍 Auto-Detect Level

Phân tích ID: FIRESWORD_1 → Level 1. Tự tìm recipe có result là FIRESWORD_2

📋 Select Menu

Quét toàn bộ inventory, liệt kê tất cả vật phẩm có công thức nâng cấp

ℹ️ Quy ước đặt tên ID: ITEMNAME_LEVEL (ví dụ: FIRESWORD_1, FIRESWORD_2, FIRESWORD_3)
🛠️ Editor GUI trong game

Quản lý toàn bộ hệ thống trao đổi trực tiếp trong game mà không cần chỉnh sửa file YAML.

📁 Group Editor

Tạo, xóa, đổi tên group. Xem danh sách category trong group.

📋 Category Editor

Tạo, xóa, đổi tên, di chuyển giữa groups, thay đổi layout.

🧪 Recipe Editor

Tạo, xóa, đổi tên, thay đổi Result, tùy chỉnh slot/page.

🧱 Ingredient Editor

Thêm, xóa, sửa nguyên liệu: số lượng, tên, AllowStorage.

📦 Tích hợp VietStorage

Khi VietStorage được cài đặt, nguyên liệu có thể được lấy từ cả inventory và kho lưu trữ.

  • Bật/tắt cho từng ingredient riêng biệt (AllowStorage: true/false)
  • Ưu tiên inventory: lấy từ inventory trước, thiếu bao nhiêu thì lấy từ kho
  • Chỉ hỗ trợ cho nguyên liệu loại VANILLA (vật phẩm Minecraft gốc)
  • Tự động phát hiện VietStorage — không cần cấu hình thêm
⚡ Auto-Create Recipes

Tạo nhanh hàng loạt recipe nâng cấp theo chuỗi level chỉ với vài thao tác trong Editor.

📋 Ví dụ: Auto-create 5 level kiếm lửa
# Chỉ cần nhập: Result Type: SWORD Result Base ID: FIRESWORD # → FIRESWORD_1, FIRESWORD_2, ... Max Level: 5 Ingredient Type: MATERIAL Ingredient ID: UPGRADE_STONE Ingredient Increment: 3 # Level 1: 3, Level 2: 6, Level 3: 9,... # Tự động tạo ra: # - FIRESWORD_1: cần 3 UPGRADE_STONE # - FIRESWORD_2: cần 6 UPGRADE_STONE + FIRESWORD_1 # - FIRESWORD_3: cần 9 UPGRADE_STONE + FIRESWORD_2 # - ...
🔢 Shift Exchange

Trao đổi nhiều lần liên tiếp với số lượng tùy chỉnh thay vì click từng lần một.

  • Shift + Right Click trên recipe: mở dialog chọn số lượng
  • Shift + Left Click trên recipe: trao đổi theo số lượng đã chọn
  • Số lượng mặc định: 5, có thể thay đổi bất cứ lúc nào
  • Số lượng được lưu vào PersistentDataContainer của player
  • Plugin tự dừng khi hết nguyên liệu hoặc đầy inventory
🎨 Custom Layout Menu

Mỗi category có thể có giao diện menu riêng biệt.

  • File layout mặc định: menu/category/default.yml
  • Tạo file mới trong menu/category/ để thêm layout
  • Gán layout cho category qua Editor hoặc file YAML (Layout: "tên_layout")
  • Drag-and-drop Recipe Layout Editor để chỉnh vị trí slot recipe
  • Recipe có thể được đặt cố định vào slot và page cụ thể
🎲 Tỉ lệ thành công

Hệ thống tỉ lệ thành công có thể cấu hình, tạo yếu tố may rủi cho gameplay.

✅ Thành công

Nguyên liệu bị mất, người chơi nhận Result item. Sound + message thành công.

❌ Thất bại

Tùy config: nguyên liệu mất hoặc trả lại. Sound + message thất bại.
