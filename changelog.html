24/03/2026
v1.0.0 — Phát hành chính thức 🎉 RELEASE

Bản phát hành đầu tiên với đầy đủ tính năng.

✨ Tính năng mới

  • 🔄 Hệ thống trao đổi hoàn chỉnh — Groups, Categories, Recipes, Ingredients
  • ⬆️ Nâng cấp vật phẩm — /vcu upgrade auto-detect level
  • 📋 Select Menu — /vcu select quét inventory tìm vật phẩm nâng cấp
  • 🛠️ Editor GUI — quản lý toàn bộ qua giao diện trong game
  • 📦 VietStorage — lấy nguyên liệu từ kho
  • 🎨 Custom Layout — layout menu riêng cho từng category
  • 🎲 Tỉ lệ thành công — cấu hình tỉ lệ + mất/giữ nguyên liệu
  • ⚡ Auto-Create — tạo nhanh chuỗi recipe nâng cấp
  • 🔢 Shift Exchange — trao đổi nhiều với số lượng tùy chỉnh
  • 💬 MiniMessage — hỗ trợ đầy đủ cho mọi text

🎮 Lệnh

  • /vcu open [path] [player] — mở menu (hỗ trợ path, target player)
  • /vcu upgrade — nâng cấp item trên tay
  • /vcu select — xem danh sách nâng cấp
  • /vcu editor — mở editor
  • /vcu reload — reload plugin

🔗 Tích hợp

  • MMOItems — hệ thống vật phẩm
  • NightCore — framework
  • VietStorage — kho lưu trữ (tùy chọn)
