Tài liệu kỹ thuật chi tiết dành cho các quản trị viên và nhà phát triển. Tìm hiểu sâu hơn về kiến trúc hệ thống, quy trình cài đặt và tùy biến giao diện VietCraftUpgrade.

Data Structure (Cấu trúc dữ liệu)

Kiến trúc phân tầng của hệ thống công thức

Hệ thống VietCraftUpgrade được xây dựng trên mô hình phân cấp 4 lớp chặt chẽ để đảm bảo khả năng mở rộng tối đa:

1

Group (Nhóm)

Lớp cao nhất, định nghĩa loại hình nâng cấp (vd: Weapon, Armor).

2

Category (Danh mục)

Phân loại nhỏ hơn trong nhóm (vd: Swords, Bows).

3

Recipe (Công thức)

Định nghĩa cụ thể các nguyên liệu và kết quả đầu ra.

4

Ingredient (Nguyên liệu)

Các item chi tiết cần thiết để thực hiện công thức.

category_swords.yml 
category_id: "swords_tier_1"
display_name: "<gradient:#ff8f73:#ffb95c>Thanh Kiếm Sơ Khai"
icon: DIAMOND_SWORD
recipes:
  - id: "sharp_sword_upgrade"
    ingredients:
      - item: IRON_INGOT
        amount: 16
      - item: DIAMOND
        amount: 2
    result: "DIAMOND_SWORD{CustomModelData:1}"

Installation Guide

Quy trình thiết lập plugin chuẩn

01

Install Dependencies

Đảm bảo bạn đã cài đặt Vault và PlaceholderAPI để hệ thống hoạt động ổn định.

02

Download JAR File

Tải phiên bản mới nhất từ trang GitHub hoặc Resource của VietCraftUpgrade.

03

Upload & Start

Tải file vào thư mục /plugins và khởi động lại server.

04

Configuration

Truy cập plugins/VietCraftUpgrade/config.yml để cấu hình cơ sở dữ liệu.
