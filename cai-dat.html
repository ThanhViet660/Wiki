Cài đặt & Khởi đầu

Bắt đầu hành trình tối ưu hóa máy chủ của bạn với hệ thống VietCraftUpgrade. Theo dõi các bước dưới đây để thiết lập môi trường hoàn hảo cho plugin.

01

Cài đặt Dependencies

inventory_2

VietCraftUpgrade yêu cầu bộ khung lõi để vận hành các tính năng nâng cao. Hãy chắc chắn bạn đã cài đặt các plugin sau:

diamond NightCore
v2.4.1+
swords MMOItems
v6.9.4+
auto_fix_high MythicLib
v1.6.2+
02

Tải về JAR

Sử dụng phiên bản ổn định nhất từ kho lưu trữ chính thức.

restart_alt
03

Khởi động lại

Sau khi copy file JAR vào thư mục /plugins, hãy khởi động lại server để tạo file config.

04

Kiểm tra cấu hình

1 database:
2 type: SQLITE
3 auto-save: true
4 debug-mode: false
05

Cấp Quyền (Permissions)

Người chơi Default

vcu.user.base

Quản trị viên Admin

vcu.admin.*

06

Thử nghiệm Lệnh

/vcu Mở menu điều khiển chính của plugin.
/vcu reload Tải lại toàn bộ tệp cấu hình.
/vcu info Kiểm tra thông tin phiên bản hiện tại.
